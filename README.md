## Notes

별도의 빌드 도구 없이 GitHub Pages에서 바로 배포하는 정적 사이트입니다.

공통 스타일은 `styles.css`, 이미지 확대 모달과 간단한 화면 동작은 `script.js`에서 관리합니다.
CSS나 JS를 수정한 뒤에는 GitHub Pages 캐시 반영을 위해 HTML에 적용된 query version도 함께 갱신합니다.
