BUIDL Asia Story Track Project

AI 학습 데이터 기여도를 추적해 원작자에게 보상을 분배하는 블록체인 기반 IP 보상 플랫폼입니다.

Overview

데이터 소유권과 보상 흐름을 정의하고 사용자 흐름을 기획했습니다. AI 학습 데이터의 기여도를 추적해 원작자에게 보상을 분배하는 구조를 제안했으며, BUIDL Asia Story Track 최우수상을 수상했습니다.

Focus

  • 데이터 소유권과 보상 흐름 정의
  • AI 학습 데이터 기여도 추적 구조 기획
  • 원작자 보상 분배 사용자 흐름 설계
