Graduation Project · 2024 ~ 2025

Zkare Medical Record Management System

블록체인 기반 의료 기록 관리 시스템입니다. 의료 기록의 접근 권한, 무결성 검증, 암호화, 분산 저장 흐름을 프로젝트 기반으로 설계하고 구현했습니다.

Project Summary

1년간 진행한 졸업 프로젝트로, Ethereum Smart Contract와 React 프론트엔드를 연동하고, 권한 기반 접근제어, ECDSA 무결성 검증, AES/RSA 이중 암호화, IPFS 연동, 환자 동의 시스템, 다중 병원 네트워크 구조를 설계했습니다.

Key Work

  • 환자 동의 기반 의료 기록 접근 흐름 설계
  • Smart Contract와 프론트엔드 연동
  • ECDSA 기반 무결성 검증 구조 구현
  • AES/RSA 이중 암호화와 IPFS 저장 흐름 설계
